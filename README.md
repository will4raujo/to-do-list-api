$ yarn install

## Start Container

Antes de rodar a aplicação, certifique-se de subir o container (por exemplo, Docker):

```bash
# Exemplo
$ docker-compose up -d
```

## Compile and Run the Project

```bash
# Desenvolvimento
$ yarn run start

# Watch mode
$ yarn run start:dev
```
